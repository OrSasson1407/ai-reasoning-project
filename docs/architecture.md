# AI Reasoning Architecture v2.0
The system is built on a 7-layer cognitive framework:
1. Ingestion (Raw Input)
2. Structuring (Claim Extraction)
3. Consistency (CCP Protocol)
4. Inference (Deductive/Inductive/Abductive)
5. World Model (Physical/Human Dynamics)
6. Meta-Cognition (Self-Reflection)
7. Oversight (Values Constraint Layer)
