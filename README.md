# AI Reasoning Project - Architecture v2.0

This repository contains the conceptual software architecture for the AI Reasoning Project, implementing a 7-layer cognitive framework.

## Strict Architectural Compliance
This codebase strictly enforces the constraints laid out in the foundational architecture documents:
* **DAD v1.0** (Data Architecture Document)
* **CCP v1.0** (Contradiction & Consistency Protocol)
* **RRS v1.0** (Reasoning Rules Specification)
* **AIC v1.0** (API & Interface Contracts)
* **EBD v1.0** (Evaluation & Benchmarking Document)

## Getting Started
1. Install dependencies: `pip install -r requirements.txt`
2. Run architectural compliance tests: `pytest`
3. Boot the Core Engine: `python run.py`
4. Boot the API Gateway Edge Enforcer: `cd api-gateway && uvicorn main:app --port 8080`
